# TurboShare - فائل شیئرنگ ویب سائٹ

ایک جدید، محفوظ اور تیز رفتار فائل شیئرنگ ویب سائٹ جو WebRTC ٹیکنالوجی استعمال کرتی ہے۔

## خصوصیات
- موبائل سے موبائل فائل شیئرنگ
- کمپیوٹر سے موبائل کنیکشن
- مکمل اینڈ-ٹو-اینڈ انکرپشن
- کوئی فائل سائز محدودیت نہیں
- کوئی سرور اسٹوریج نہیں
- ریئل ٹائم ٹرانسفر

## Deployment Steps

### 1. GitHub پر اپلوڈ کریں
1. نیا ریپوزٹری بنائیں
2. سارے فائلیں اپلوڈ کریں
3. Settings > Pages میں جائیں
4. Branch کو `main` پر سیٹ کریں
5. Save پر کلک کریں

### 2. بیک اینڈ سرور سیٹ اپ
1. Node.js انسٹال کریں
2. اپنے ہوسٹنگ پر فائلیں اپلوڈ کریں
3. SSH کنیکٹ کریں:

npm install express socket.io
